appearance

Definition

The appearance CSS property is used to control the appearance of UI elements to match a native UI theme.

Values

Value Description
none Removes any native styling from the element, applying only the specified CSS styles.
auto Applies a platform-native styling based on the user agent.
button Makes the element look like a button.
textfield Makes the element look like a text field.
searchfield Makes the element look like a search field.
textarea Makes the element look like a textarea.
checkbox Makes the element look like a checkbox.
radio Makes the element look like a radio button.
menuitem Makes the element look like a menu item.
menulist Makes the element look like a menu list.
listbox Makes the element look like a list box.
meter Makes the element look like a meter.
progress-bar Makes the element look like a progress bar.
slider Makes the element look like a slider.
spinbutton Makes the element look like a spin button.
combobox Makes the element look like a combo box.

Description

The appearance property allows developers to make elements appear like standard native UI controls. This can be useful for creating consistent and familiar user interfaces. By default, some elements are rendered with their native styling, which may vary across browsers and operating systems. The appearance property helps to standardize this styling.

All Most Similar Properties

Property Description Difference
display Controls how an element is displayed on the screen Affects the rendering mode, not the visual style
visibility Controls whether an element is visible or not Only affects visibility, not the appearance
background Sets the background color or image of an element Affects the background, but not the native UI styling
border Sets the border style, width, and color of an element Affects the border, not the overall native appearance
outline Sets the outline style, width, and color of an element Affects the outline, not the overall native appearance
