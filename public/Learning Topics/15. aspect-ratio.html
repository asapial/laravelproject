aspect-ratio

Definition

The aspect-ratio CSS property sets a preferred aspect ratio for an element, which is the ratio between the width and the height of the element.

Values

Value Description
<ratio> Specifies the aspect ratio in the form of width / height.
auto The intrinsic aspect ratio of the element is used if available.

Example

.example-16-9 {
  aspect-ratio: 16 / 9; /* Sets the aspect ratio to 16:9 */
}

.example-4-3 {
  aspect-ratio: 4 / 3; /* Sets the aspect ratio to 4:3 */
}

.example-1-1 {
  aspect-ratio: 1 / 1; /* Sets the aspect ratio to 1:1 (square) */
}

.example-auto {
  aspect-ratio: auto; /* Uses the intrinsic aspect ratio */
}

Description

The aspect-ratio property defines the width-to-height ratio of an element, helping maintain a consistent layout and preventing distortion. This is particularly useful for responsive design, ensuring elements scale proportionally across different screen sizes.

All Most Similar Properties

Property Description Difference
width Sets the width of an element Does not maintain a proportional relationship with height
height Sets the height of an element Does not maintain a proportional relationship with width
padding Sets the padding inside an element Can be used to maintain aspect ratio using padding trick, but not as direct
min-width Sets the minimum width of an element Does not maintain a proportional relationship with height
min-height Sets the minimum height of an element Does not maintain a proportional relationship with width
