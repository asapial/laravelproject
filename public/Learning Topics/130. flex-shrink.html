flex-shrink

Definition

The flex-shrink CSS property specifies how much a flex item will shrink relative to the rest of the flex items inside the same container when there isn't enough space.

Values

Value Description
0 The item will not shrink.
<number> A number greater than 0 that specifies the item's shrinking factor.
initial Sets the property to its default value 1.
inherit Inherits the property from its parent element.

Example

/* No shrink */
.item {
    flex-shrink: 0; /* Item will not shrink */
}

/* Equal shrink */
.item {
    flex-shrink: 1; /* Item will shrink equally */
}

/* Double shrink */
.item {
    flex-shrink: 2; /* Item will shrink twice as much as other items */
}

Description

The flex-shrink property is part of the flexible box layout module. It determines how a flex item will shrink relative to the other items inside the same flex container when space is insufficient. If all items have the same flex-shrink value, they will shrink at the same rate. If one item has a higher flex-shrink value, it will shrink more compared to the other items.

All Most Similar Properties

Property Description Difference
flex-grow Specifies how a flex item will grow. flex-grow determines growth when space is available, while flex-shrink determines how items shrink when space is insufficient.
flex-basis Specifies the initial main size of a flex item. flex-basis sets the initial size of the item before growing or shrinking, while flex-shrink controls shrinking.
flex Shorthand for flex-grow, flex-shrink, and flex-basis. The flex property is a shorthand for setting all three properties at once.

In JavaScript

// Set flex-shrink property using JavaScript
document.querySelector('.item').style.flexShrink = '2';
