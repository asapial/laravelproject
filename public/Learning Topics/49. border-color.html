border-color

Definition

The border-color CSS property sets the color of an element's border.

Values

Value Description
<color> Specifies the border color. Can be a named color, hex, rgb, rgba, hsl, or hsla value.
inherit Inherits the border color from its parent element.
initial Sets the property to its default value.
unset Resets the property to its natural value, either inherited or initial.

Example

/* Different border-color values */
.example1 {
    border-color: red; /* Named color */
}

.example2 {
    border-color: #00FF00; /* Hex color */
}

.example3 {
    border-color: rgb(0, 0, 255); /* RGB color */
}

.example4 {
    border-color: rgba(255, 165, 0, 0.5); /* RGBA color with transparency */
}

.example5 {
    border-color: hsl(120, 100%, 50%); /* HSL color */
}

.example6 {
    border-color: hsla(240, 100%, 50%, 0.3); /* HSLA color with transparency */
}

.example7 {
    border-color: inherit; /* Inherited color */
}

.example8 {
    border-color: initial; /* Initial color */
}

.example9 {
    border-color: unset; /* Unset color */
}

Description

The border-color property sets the color of an element's border. This can be done using various color formats like named colors, hexadecimal values, RGB, RGBA, HSL, and HSLA. The property also supports keyword values such as inherit, initial, and unset to control inheritance and default behaviors. This property can set one color for all four borders or different colors for each border by specifying multiple values.

All Most Similar Properties

Property Description Difference
border-top-color Sets the color of the top border of an element. Applies only to the top border.
border-right-color Sets the color of the right border of an element. Applies only to the right border.
border-bottom-color Sets the color of the bottom border of an element. Applies only to the bottom border.
border-left-color Sets the color of the left border of an element. Applies only to the left border.
border A shorthand property for setting border width, style, and color. Applies to all aspects of the border.
