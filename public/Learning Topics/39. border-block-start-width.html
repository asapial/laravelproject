border-block-start-width

Definition

The border-block-start-width CSS property sets the width of the logical block-start border of an element, which corresponds to the top border in a horizontal writing mode.

Values

Value Description
<length> Specifies the width of the border using a length unit (px, em, rem, etc.).
thin A thin border width.
medium A medium border width.
thick A thick border width.
inherit Inherits the border width from its parent element.
initial Sets the border width to its default value.
unset Resets the border width to its inherited or initial value.

Example

/* <style> value */
.element1 {
  border-block-start-width: 2px; /* Sets the border width to 2 pixels */
}

.element2 {
  border-block-start-width: 0.5em; /* Sets the border width to 0.5 em */
}

.element3 {
  border-block-start-width: thin; /* Sets the border width to thin */
}

.element4 {
  border-block-start-width: medium; /* Sets the border width to medium */
}

.element5 {
  border-block-start-width: thick; /* Sets the border width to thick */
}

.element6 {
  border-block-start-width: inherit; /* Inherits the border width from the parent */
}

.element7 {
  border-block-start-width: initial; /* Sets the border width to its default value */
}

.element8 {
  border-block-start-width: unset; /* Resets the border width to its inherited or default value */
}

Description

The border-block-start-width property is part of the CSS Logical Properties and Values module, providing control over layout in a writing mode-independent manner. This property allows for setting the width of the block-start border of an element, accommodating different writing modes and text orientations. It enhances flexibility in design by addressing the logical flow of content, rather than fixed physical directions.

All Most Similar Properties

Property Description Difference
border-top-width Sets the width of the top border. Physical property, always targets the top border regardless of writing mode.
border-block-end-width Sets the width of the block-end border. Targets the block-end border, corresponds to the bottom border in a horizontal mode.
border-inline-start-width Sets the width of the inline-start border. Targets the inline-start border, which corresponds to the left border in LTR mode.
border-inline-end-width Sets the width of the inline-end border. Targets the inline-end border, which corresponds to the right border in LTR mode.
