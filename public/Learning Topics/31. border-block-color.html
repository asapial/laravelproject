border-block-color

Definition

The border-block-color CSS property sets the color of an element's block-start and block-end borders.

Values

Value Description
color Specifies the color of the block-start and block-end borders. Can be any valid color value (e.g., name, hex, rgb, rgba, hsl, hsla).

Example

/* Using color names */
border-block-color: red; /* Red color for both block-start and block-end borders */

/* Using hex values */
border-block-color: #00ff00; /* Green color for both block-start and block-end borders */

/* Using RGB values */
border-block-color: rgb(0, 0, 255); /* Blue color for both block-start and block-end borders */

/* Using RGBA values */
border-block-color: rgba(255, 0, 0, 0.5); /* Semi-transparent red color for both block-start and block-end borders */

/* Using HSL values */
border-block-color: hsl(120, 100%, 50%); /* Green color for both block-start and block-end borders */

/* Using HSLA values */
border-block-color: hsla(240, 100%, 50%, 0.5); /* Semi-transparent blue color for both block-start and block-end borders */

Description

The border-block-color property is used to set the color of the block-start and block-end borders of an element. In writing modes where the block direction is vertical, these edges correspond to the top and bottom edges of the element. The property can be used to specify the color using any valid CSS color value, providing flexibility to match various design requirements.

All Most Similar Properties

Property Description Difference
border-color Sets the color of all four borders of an element. Affects all four borders, not just the block borders.
border-top-color Sets the color of the top border of an element. Only affects the top border.
border-bottom-color Sets the color of the bottom border of an element. Only affects the bottom border.
border-block-start-color Sets the color of the block-start border of an element. Only affects the block-start border.
border-block-end-color Sets the color of the block-end border of an element. Only affects the block-end border.
