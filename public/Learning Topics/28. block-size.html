block-size

Definition

The block-size CSS property defines the size of an element in the block direction, which is determined by the writing mode. In horizontal writing modes, it corresponds to the height, and in vertical writing modes, it corresponds to the width.

Values

Value Description
<length> Specifies the block size using length units such as px, em, rem, etc.
<percentage> Specifies the block size as a percentage of the containing block's block size.
auto The browser calculates the block size.
max-content The intrinsic preferred block size.
min-content The intrinsic minimum block size.
fit-content The maximum size is the available space, the minimum is the intrinsic minimum size.
inherit Inherits this property from its parent element.
initial Sets this property to its default value.
unset Resets the property to its natural value, which is either inherit or initial.

Example

/* Using length values */
block-size: 100px; /* 100 pixels block size */
block-size: 50em; /* 50 ems block size */

/* Using percentage values */
block-size: 50%; /* 50% of the containing block's block size */

/* Using keyword values */
block-size: auto; /* Browser calculates the block size */
block-size: max-content; /* Intrinsic preferred block size */
block-size: min-content; /* Intrinsic minimum block size */
block-size: fit-content; /* Fits content without overflowing */

Description

The block-size property in CSS sets the size of an element in the block direction, which varies depending on the writing mode. In most cases, for horizontal writing modes, the block direction corresponds to the element's height, while for vertical writing modes, it corresponds to the element's width. This property allows for a flexible and consistent approach to setting dimensions that adapt to different writing modes. The values can be specified using length units, percentages, or special keywords that handle intrinsic sizing.

All Most Similar Properties

Property Description Difference
width Sets the width of an element. Always corresponds to the inline dimension.
height Sets the height of an element. Always corresponds to the block dimension in horizontal writing modes.
inline-size Sets the size of an element in the inline direction. Depends on the writing mode, typically the width in horizontal writing modes.
min-height Sets the minimum height of an element. Affects only the height.
max-height Sets the maximum height of an element. Affects only the height.
min-width Sets the minimum width of an element. Affects only the width.
max-width Sets the maximum width of an element. Affects only the width.
