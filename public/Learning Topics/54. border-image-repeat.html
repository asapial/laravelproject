border-image-repeat

Definition

The border-image-repeat CSS property specifies how the edge regions of the border image are adjusted to fit the dimensions of the border image area.

Values

Value Description
stretch The border image is stretched to fill the area.
repeat The border image is tiled (repeated) to fill the area.
round The border image is tiled (repeated) to fill the area, and if it does not fit perfectly, the image is rescaled so that it does.
space The border image is tiled (repeated) to fill the area, and extra space is distributed around the tiles.

Example

/* Example with stretch value */
.example-stretch {
  border-image-repeat: stretch; /* Stretches the border image to fill the area */
}

/* Example with repeat value */
.example-repeat {
  border-image-repeat: repeat; /* Tiles the border image to fill the area */
}

/* Example with round value */
.example-round {
  border-image-repeat: round; /* Tiles the border image and resizes it to fit perfectly */
}

/* Example with space value */
.example-space {
  border-image-repeat: space; /* Tiles the border image and distributes extra space around the tiles */
}

Description

The border-image-repeat property controls how the slices of a border image are scaled and tiled. It can be used to define whether the border image should be stretched, repeated, rounded, or spaced to fit the border area. This property helps in creating visually appealing and flexible border designs.

All Most Similar Properties

Property Description Difference
border-image-width Sets the width of the border image area. border-image-width controls the thickness of the border image; border-image-repeat controls the tiling method.
border-image-outset Specifies the amount by which the border image area extends beyond the border box. border-image-outset extends the border image area; border-image-repeat defines the tiling method.
border-image-slice Specifies how to slice the border image. border-image-slice defines the inward slices of the border image; border-image-repeat defines the outward tiling.
