border-collapse

Definition

The border-collapse CSS property sets whether the borders of a table are collapsed into a single border or detached as in standard HTML.

Values

Value Description
collapse Borders are collapsed into a single border, eliminating the space between cells.
separate Borders are separated, maintaining the space between cells as defined by the border-spacing property.

Example

/* Different border-collapse values */
.table1 {
    border-collapse: collapse; /* Collapsed borders */
}

.table2 {
    border-collapse: separate; /* Separate borders */
}

Description

The border-collapse property controls the table's border model. When set to collapse, adjacent table cell borders are merged into a single border, removing any gaps between cells. When set to separate, the table retains individual cell borders, allowing for gaps as specified by the border-spacing property.

All Most Similar Properties

Property Description Difference
border-spacing Sets the distance between the borders of adjacent table cells. Only affects tables with border-collapse: separate.
table-layout Defines the algorithm used to lay out the table cells, rows, and columns. Affects the overall table layout, not border behavior.
