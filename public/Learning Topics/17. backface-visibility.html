backface-visibility

Definition

The backface-visibility CSS property determines whether the back face of an element is visible when the element is rotated.

Values

Value Description
visible The back face is visible when the element is rotated.
hidden The back face is not visible when the element is rotated.

Example

/* Example values for backface-visibility */
.element-visible {
  backface-visibility: visible; /* The back face is visible */
}

.element-hidden {
  backface-visibility: hidden; /* The back face is not visible */
}

Description

The backface-visibility property is used in 3D transformations to control whether the back face of an element is visible when it is rotated. This is particularly useful when you want to hide the content of the back face to avoid displaying a mirrored or reversed version of the front face.

All Most Similar Properties

Property Description Difference
visibility Specifies whether the element is visible or not. visibility controls the visibility of the entire element, not just the back face.
display Specifies the display behavior of an element (none, block, inline, etc.). display removes the element from the document flow when set to none, whereas backface-visibility only affects the visibility of the back face.

        .container {
            width: 80%;

        }

            position: relative;
            width: 100%;
            height: 200px;
            text-align: center;
            transition: transform 0.6s;
            transform-style: preserve-3d;
        }
        .sub-container:hover .flip-box {
            transform: rotateY(180deg);

            color: black;

        }
        .visible .flip-box-back {
            backface-visibility: visible;
        }
        .hidden .flip-box-back {
            backface-visibility: hidden;
        }
