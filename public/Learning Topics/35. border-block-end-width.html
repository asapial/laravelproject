border-block-end-width

Definition

The border-block-end-width CSS property sets the width of the logical block-end border of an element, which corresponds to the bottom border in a horizontal writing mode.

Values

Value Description
<length> Specifies the width of the border using a length value.
thin Sets the border to a thin width.
medium Sets the border to a medium width (default).
thick Sets the border to a thick width.
inherit Inherits the width from its parent element.
initial Sets the width to its default value.
unset Resets the property to either its inherited or initial value.

Example

/* <length> value */
.element1 {
  border-block-end-width: 5px; /* Sets the border width to 5px */
}

/* thin value */
.element2 {
  border-block-end-width: thin; /* Sets the border width to thin */
}

/* medium value */
.element3 {
  border-block-end-width: medium; /* Sets the border width to medium */
}

/* thick value */
.element4 {
  border-block-end-width: thick; /* Sets the border width to thick */
}

/* inherit value */
.element5 {
  border-block-end-width: inherit; /* Inherits the border width from the parent */
}

/* initial value */
.element6 {
  border-block-end-width: initial; /* Sets the border width to its default value */
}

/* unset value */
.element7 {
  border-block-end-width: unset; /* Resets the border width to its inherited or default value */
}

Description

The border-block-end-width property is part of the CSS Logical Properties and Values module, which provides the ability to control layout through logical, rather than physical, direction and dimension mappings. This property allows for more flexible and intuitive design for different writing modes and orientations. It specifically sets the width of the block-end border of an element, which typically corresponds to the bottom border in left-to-right, top-to-bottom writing modes.

All Most Similar Properties

Property Description Difference
border-bottom-width Sets the width of the bottom border of an element. Physical property, always targets the bottom border regardless of writing mode.
border-block-start-width Sets the width of the logical block-start border. Targets the block-start border, corresponds to the top border in a horizontal mode.
border-inline-start-width Sets the width of the logical inline-start border. Targets the inline-start border, which corresponds to the left border in LTR mode.
border-inline-end-width Sets the width of the logical inline-end border. Targets the inline-end border, which corresponds to the right border in LTR mode.
