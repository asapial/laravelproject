border-block

Definition

The border-block CSS property is a shorthand property that defines the border properties for the block start and block end edges of an element. It is used in place of the separate border-block-start and border-block-end properties.

Values

Value Description
border-width Specifies the width of the border. Possible values: thin, medium, thick, or any valid length value (e.g., px, em).
border-style Specifies the style of the border. Possible values: none, hidden, dotted, dashed, solid, double, groove, ridge, inset, outset.
border-color Specifies the color of the border. Can be any valid color value (e.g., name, hex, rgb, rgba, hsl, hsla).

Example

/* Using keyword values for width, style, and color */
border-block: thin solid black; /* Thin solid black border */
border-block: medium dashed red; /* Medium dashed red border */
border-block: thick dotted blue; /* Thick dotted blue border */

/* Using length values for width and different styles and colors */
border-block: 1px solid #000; /* 1px solid black border */
border-block: 2em double rgba(255, 0, 0, 0.5); /* 2em double semi-transparent red border */
border-block: 5px groove hsl(120, 100%, 50%); /* 5px groove green border */

Description

The border-block property is a shorthand property in CSS that sets the width, style, and color of an element's border on the block start and block end edges. In writing modes where the block direction is vertical, these edges correspond to the top and bottom edges of the element. The property is a shorthand for setting border-block-start and border-block-end in a single declaration, and it simplifies the styling of borders in various writing modes.

All Most Similar Properties

Property Description Difference
border-top Sets the border on the top edge of an element. Only affects the top edge, irrespective of writing mode.
border-bottom Sets the border on the bottom edge of an element. Only affects the bottom edge, irrespective of writing mode.
border-block-start Sets the border on the block start edge of an element. Only affects the block start edge.
border-block-end Sets the border on the block end edge of an element. Only affects the block end edge.
border Sets the border on all four edges of an element. Affects all edges, not just the block edges.
