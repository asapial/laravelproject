animation-duration

Definition

The animation-duration CSS property specifies the length of time that an animation should take to complete one cycle.

Values

Value Description
<time> Specifies the duration of the animation. The value can be specified in seconds (s) or milliseconds (ms).

Example

.item-duration-1s {
    animation-duration: 1s; /* 1 second duration */
}

.item-duration-2s {
    animation-duration: 2s; /* 2 second duration */
}

.item-duration-500ms {
    animation-duration: 500ms; /* 500 milliseconds duration */
}

Description

The animation-duration property defines how long an animation takes to complete one cycle. It can be specified in seconds (s) or milliseconds (ms). By controlling the duration, you can make animations appear faster or slower, depending on the desired effect. This property is a crucial part of creating smooth and engaging animations in web design.

All Most Similar Properties

Property Description Difference
transition-duration Specifies the length of time a transition effect takes to complete. transition-duration is used for transitions, while animation-duration is used for animations.
animation-delay Specifies the delay before the animation starts. animation-delay defines when an animation starts, while animation-duration defines how long it takes.
