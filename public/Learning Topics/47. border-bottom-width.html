border-bottom-width

Definition

The border-bottom-width CSS property sets the width of the bottom border of an element.

Values

Value Description
<length> Specifies the width using a length value (e.g., px, em).
thin A predefined thin border width.
medium A predefined medium border width.
thick A predefined thick border width.
inherit Inherits the border width from its parent element.
initial Sets the property to its default value.
unset Resets the property to its natural value.

Example

/* Different border-bottom widths */
.example1 {
    border-bottom-width: 1px; /* 1 pixel width */
}

.example2 {
    border-bottom-width: 0.5em; /* 0.5 em width */
}

.example3 {
    border-bottom-width: 10px; /* 10 pixels width */
}

.example4 {
    border-bottom-width: thin; /* thin predefined width */
}

.example5 {
    border-bottom-width: medium; /* medium predefined width */
}

.example6 {
    border-bottom-width: thick; /* thick predefined width */
}

.example7 {
    border-bottom-width: inherit; /* inherits the width from parent */
}

.example8 {
    border-bottom-width: initial; /* sets the property to its default value */
}

.example9 {
    border-bottom-width: unset; /* resets the property to its natural value */
}

Description

The border-bottom-width property defines the thickness of the bottom border of an element. It can be specified using length units (such as pixels, ems, etc.), or using predefined values like thin, medium, and thick. The property also supports keyword values such as inherit, initial, and unset to control inheritance and default behaviors.

All Most Similar Properties

Property Description Difference
border-top-width Sets the width of the top border of an element. Applies to the top border.
border-left-width Sets the width of the left border of an element. Applies to the left border.
border-right-width Sets the width of the right border of an element. Applies to the right border.
border-width A shorthand property for setting the width of all four borders. Applies to all borders simultaneously.
border A shorthand property for setting border width, style, and color. Applies to all aspects of the border.
