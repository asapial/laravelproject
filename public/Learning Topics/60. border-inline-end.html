border-inline-end

Definition

The border-inline-end CSS property is a shorthand property for setting the width, style, and color of an element's logical inline-end border.

Values

Value Description
<'border-width'> Specifies the width of the inline-end border.
<'border-style'> Specifies the style of the inline-end border.
<'border-color'> Specifies the color of the inline-end border.
inherit Inherits the value from its parent element.
initial Sets the property to its default value.
unset Resets the property to its inherited value if it inherits, or to its initial value if it does not inherit.

Example

/* Example with specific width, style, and color */
.example-specific {
  border-inline-end: 5px solid blue; /* Sets the inline-end border width to 5px, style to solid, and color to blue */
}

/* Example with inherit value */
.example-inherit {
  border-inline-end: inherit; /* Inherits the value from the parent element */
}

/* Example with initial value */
.example-initial {
  border-inline-end: initial; /* Sets the property to its default value */
}

/* Example with unset value */
.example-unset {
  border-inline-end: unset; /* Resets the property to its inherited or initial value */
}

Description

The border-inline-end property is a shorthand property that allows you to define the border properties for the inline-end border of an element in a single declaration. This property takes into account the writing mode and text directionality of the element, applying the styles to the end side of the inline direction.

All Most Similar Properties

Property Description Difference
border Shorthand for setting border-width, border-style, and border-color on all four sides of an element. border applies to all sides, while border-inline-end applies to the inline-end border only.
border-inline Shorthand for setting border-inline-start and border-inline-end. border-inline targets both inline-start and inline-end borders, while border-inline-end targets only the inline-end border.
border-inline-start Sets the border properties for the inline-start edge. border-inline-start targets the start edge, while border-inline-end targets the end edge.
