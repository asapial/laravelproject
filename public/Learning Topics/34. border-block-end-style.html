border-block-end-style

Definition

The border-block-end-style CSS property sets the style of the logical block-end border of an element, which corresponds to the bottom border in a horizontal writing mode.

Values

Value Description
none No border.
hidden Same as none, but with different border conflict resolution for table elements.
dotted A series of round dots.
dashed A series of square-ended dashes.
solid A single solid line.
double Two solid lines. The sum of the two lines and the space between them equals the border width.
groove Looks as if it were carved in the canvas.
ridge Looks as if it were coming out of the canvas.
inset Looks as if the content is embedded in the canvas.
outset Looks as if the content is coming out of the canvas.
inherit Inherits the style from its parent element.
initial Sets the style to its default value.
unset Resets the property to either its inherited value or its initial value.

Example

/* none value */
.element1 {
  border-block-end-style: none; /* No border */
}

/* hidden value */
.element2 {
  border-block-end-style: hidden; /* No border, but different behavior in tables */
}

/* dotted value */
.element3 {
  border-block-end-style: dotted; /* Dotted border */
}

/* dashed value */
.element4 {
  border-block-end-style: dashed; /* Dashed border */
}

/* solid value */
.element5 {
  border-block-end-style: solid; /* Solid border */
}

/* double value */
.element6 {
  border-block-end-style: double; /* Double border */
}

/* groove value */
.element7 {
  border-block-end-style: groove; /* Grooved border */
}

/* ridge value */
.element8 {
  border-block-end-style: ridge; /* Ridged border */
}

/* inset value */
.element9 {
  border-block-end-style: inset; /* Inset border */
}

/* outset value */
.element10 {
  border-block-end-style: outset; /* Outset border */
}

/* inherit value */
.element11 {
  border-block-end-style: inherit; /* Inherits the border style from the parent */
}

/* initial value */
.element12 {
  border-block-end-style: initial; /* Sets the border style to its default value */
}

/* unset value */
.element13 {
  border-block-end-style: unset; /* Resets the border style to its inherited or default value */
}

Description

The border-block-end-style property is part of the CSS Logical Properties and Values module, which provides the ability to control layout through logical, rather than physical, direction and dimension mappings. This property allows for more flexible and intuitive design for different writing modes and orientations. It specifically sets the style of the block-end border of an element, which typically corresponds to the bottom border in left-to-right, top-to-bottom writing modes.

All Most Similar Properties

Property Description Difference
border-bottom-style Sets the style of the bottom border of an element. Physical property, always targets the bottom border regardless of writing mode.
border-block-start-style Sets the style of the logical block-start border. Targets the block-start border, corresponds to the top border in a horizontal mode.
border-inline-start-style Sets the style of the logical inline-start border. Targets the inline-start border, which corresponds to the left border in LTR mode.
border-inline-end-style Sets the style of the logical inline-end border. Targets the inline-end border, which corresponds to the right border in LTR mode.
