animation-delay

Definition

The animation-delay CSS property specifies the amount of time to wait from applying the animation to the element before beginning to perform the animation.

Values

Value Description
<time> Specifies the delay before the animation starts. This can be a positive or negative number. A positive value delays the start of the animation, while a negative value causes the animation to start immediately as if it had already been playing for that amount of time.

Example

.item-delay-1s {
    animation-delay: 1s; /* 1 second delay */
}

.item-delay-2s {
    animation-delay: 2s; /* 2 second delay */
}

.item-delay-negative {
    animation-delay: -1s; /* Starts immediately, 1 second into the animation */
}

Description

The animation-delay property defines when an animation starts. The value is defined in seconds (s) or milliseconds (ms). If the value is positive, the animation will start after the specified time. If the value is negative, the animation will start immediately, but as if it had already been running for that duration. This is useful for creating staggered animations.

All Most Similar Properties

Property Description Difference
transition-delay Specifies when the transition effect will start. transition-delay is used for transitions, while animation-delay is used for animations.
animation-duration Specifies the duration of the animation. animation-duration defines how long an animation takes, while animation-delay defines when it starts.

Example Code

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <style>
        body {
            background-color: #f0f0f0;
            font-family: Arial, sans-serif;
        }

        .container {
            display: flex;
            flex-direction: column;
            align-items: center;
            box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1);
            padding: 20px;
            border-radius: 10px;
            margin: 20px auto;
            background-color: #fff;
            width: 80%;
        }

        .item {
            width: 100px;
            height: 100px;
            margin: 20px;
            border-radius: 5px;
            background-color: #4caf50;
        }

        @keyframes move {
            0% { transform: translateX(0); }
            100% { transform: translateX(100px); }
        }

        .item-delay-1s {
            animation: move 2s linear infinite;
            animation-delay: 1s; /* 1 second delay */
        }

        .item-delay-2s {
            animation: move 2s linear infinite;
            animation-delay: 2s; /* 2 second delay */
        }

        .item-delay-negative {
            animation: move 2s linear infinite;
            animation-delay: -1s; /* Starts immediately, 1 second into the animation */
        }

        h1 {
            text-align: center;
        }
    </style>
    <title>Animation Delay Property Example</title>
</head>
<body>
    <h1>animation-delay </h1>
    <div class="container">
        <div class="item item-delay-1s"></div>
        <div class="item item-delay-2s"></div>
        <div class="item item-delay-negative"></div>
    </div>
</body>
</html>
