border-block-start

Definition

The border-block-start CSS property is a shorthand property that sets the values for the logical block-start border width, style, and color of an element. The block-start border corresponds to the top border in a horizontal writing mode.

Values

Value Description
<line-width> <line-style> <color> Shorthand for setting border width, style, and color.
inherit Inherits the border properties from its parent element.
initial Sets the border properties to their default values.
unset Resets the border properties to either their inherited value or their initial value.

Example

/* <line-width> <line-style> <color> value */
.element1 {
  border-block-start: 5px solid black; /* Sets width to 5px, style to solid, color to black */
}

/* inherit value */
.element2 {
  border-block-start: inherit; /* Inherits the border properties from the parent */
}

/* initial value */
.element3 {
  border-block-start: initial; /* Sets the border properties to their default values */
}

/* unset value */
.element4 {
  border-block-start: unset; /* Resets the border properties to their inherited or default values */
}

Description

The border-block-start property is part of the CSS Logical Properties and Values module, which provides the ability to control layout through logical, rather than physical, direction and dimension mappings. This property allows for more flexible and intuitive design for different writing modes and orientations. It sets the width, style, and color of the block-start border of an element, which typically corresponds to the top border in left-to-right, top-to-bottom writing modes.

All Most Similar Properties

Property Description Difference
border-top Sets the width, style, and color of the top border. Physical property, always targets the top border regardless of writing mode.
border-block-end Sets the width, style, and color of the block-end border. Targets the block-end border, corresponds to the bottom border in a horizontal mode.
border-inline-start Sets the width, style, and color of the inline-start border. Targets the inline-start border, which corresponds to the left border in LTR mode.
border-inline-end Sets the width, style, and color of the inline-end border. Targets the inline-end border, which corresponds to the right border in LTR mode.
