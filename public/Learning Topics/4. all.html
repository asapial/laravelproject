all

Definition

The all CSS property resets all properties, apart from unicode-bidi and direction, to their initial or inherited values. This is useful for applying a CSS reset or for ensuring that an element is free of any inherited styles.

Values

Value Description
initial Resets all properties to their initial values.
inherit Inherits the values from the parent element.
unset Resets all properties to their inherited value if they inherit from the parent, or to their initial value if they do not inherit.

Example

.reset-initial {
    all: initial; /* Resets all properties to initial values */
}

.reset-inherit {
    all: inherit; /* Inherits all properties from the parent element */
}

.reset-unset {
    all: unset; /* Resets properties to inherited value if inheritable, otherwise to initial value */
}

Description

The all property is a shorthand that allows authors to reset all CSS properties (except unicode-bidi and direction) for an element to a specified value. This can be particularly useful for undoing the styles applied by previous rules or to enforce a certain styling starting point.

All Most Similar Properties

Property Description Difference
initial Sets the property to its default value as defined in the CSS specification. initial affects individual properties, whereas all: initial resets all properties simultaneously.
inherit Sets the property to its inherited value from its parent element. inherit affects individual properties, whereas all: inherit inherits all properties simultaneously.
unset Resets the property to its inherited value if it inherits, otherwise to its initial value. unset affects individual properties, whereas all: unset resets all properties simultaneously.
