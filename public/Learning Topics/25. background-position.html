background-position

Definition

The background-position CSS property sets the initial position for each background image. The position is relative to the position layer set by background-origin.

Values

Value Description
left top Aligns the background image to the left and top of the container.
left center Aligns the background image to the left and center of the container.
left bottom Aligns the background image to the left and bottom of the container.
right top Aligns the background image to the right and top of the container.
right center Aligns the background image to the right and center of the container.
right bottom Aligns the background image to the right and bottom of the container.
center top Aligns the background image to the center and top of the container.
center center Aligns the background image to the center and center of the container.
center bottom Aligns the background image to the center and bottom of the container.
x% y% Aligns the background image to the x% and y% position of the container.
xpos ypos Aligns the background image to the xpos and ypos position of the container.
initial Sets this property to its default value.
inherit Inherits this property from its parent element.

Example

/* Using keyword values */
background-position: left top; /* top left corner */
background-position: center center; /* center of the container */

/* Using percentage values */
background-position: 50% 50%; /* center of the container */
background-position: 100% 0%; /* top right corner */

/* Using length values */
background-position: 10px 20px; /* 10px from left, 20px from top */
background-position: 0 0; /* top left corner */

/* Using combination */
background-position: left 10px bottom 20px; /* 10px from left, 20px from bottom */

Description

The background-position property in CSS sets the starting position of a background image. By default, a background image is placed at the top-left corner of an element and repeated both vertically and horizontally. This property allows you to precisely control where a background image is placed within its container. You can specify the position in length values (pixels, ems), percentage values, or keyword values (top, bottom, left, right, center). Multiple backgrounds can have different positions set by using a comma-separated list of values. If only one value is provided, the second value defaults to center.

All Most Similar Properties

Property Description Difference
background-image Specifies one or more background images for an element. Defines the image itself rather than its position.
background-repeat Sets how background images are repeated. Controls repetition of background, not its position.
background-size Specifies the size of the background images. Defines size of background, not its starting position.
background-attachment Sets whether a background image scrolls with the page or is fixed. Controls scroll behavior, not initial position.
background-origin Specifies the positioning area of the background images. Defines area of positioning, not the actual position.
