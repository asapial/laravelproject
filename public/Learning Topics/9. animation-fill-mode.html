animation-fill-mode

Definition

The animation-fill-mode CSS property specifies how a CSS animation should apply styles to its target before and after it is executing.

Values

Value Description
none The animation will not apply any styles to the target before or after it is executing.
forwards The animation will apply the styles defined in the last keyframe (100%) after the animation has finished.
backwards The animation will apply the styles defined in the first keyframe (0%) before the animation starts.
both The animation will follow the rules for both forwards and backwards.

Example

/* none */
.example-none {
  animation-fill-mode: none; /* No styles applied before or after animation */
}

/* forwards */
.example-forwards {
  animation-fill-mode: forwards; /* Styles from last keyframe applied after animation */
}

/* backwards */
.example-backwards {
  animation-fill-mode: backwards; /* Styles from first keyframe applied before animation */
}

/* both */
.example-both {
  animation-fill-mode: both; /* Styles from first keyframe before and last keyframe after animation */
}

Description

The animation-fill-mode property determines the style applied to an element when the animation is not playing (before it starts, after it ends, or both). This can be useful for maintaining the final state of an animation or ensuring a specific style is applied immediately before an animation begins. This property enhances control over the visual state of an element across the different phases of its animation lifecycle.

All Most Similar Properties

Property Description Difference
animation-direction Specifies whether an animation should play in reverse. Controls the direction of the animation, not its styles before/after execution.
animation-iteration-count Specifies the number of times an animation should play. Controls how many times the animation plays, not the styles before/after execution.

Example Code

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Animation Fill Mode Example</title>
  <style>
    body {
      display: flex;
      justify-content: center;
      align-items: center;
      height: 100vh;
      background-color: #f0f0f0;
      margin: 0;
    }
    .container {
      width: 80%;
      padding: 20px;
      box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1);
      background-color: white;
      border-radius: 20px;
      text-align: center;
    }
    .sub-container {
      width: 80%;
      margin: 20px auto;
      padding: 10px;
      box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1);
      border-radius: 20px;
      background-color: #e9e9e9;
    }
    .box {
      width: 100px;
      height: 100px;
      margin: 20px;
      display: inline-block;
      animation: example-animation 2s infinite;
    }
    .box-none {
      animation-fill-mode: none;
      background-color: red;
    }
    .box-forwards {
      animation-fill-mode: forwards;
      background-color: blue;
    }
    .box-backwards {
      animation-fill-mode: backwards;
      background-color: green;
    }
    .box-both {
      animation-fill-mode: both;
      background-color: yellow;
    }
    @keyframes example-animation {
      0% {
        transform: translateY(0);
      }
      100% {
        transform: translateY(50px);
      }
    }
  </style>
</head>
<body>

      <div class="box box-backwards"></div>
    </div>

      <div class="box box-both"></div>
    </div>
  </div>
</body>
