align-self

Definition

The align-self CSS property allows the default alignment (or the one specified by align-items) to be overridden for individual flex items. It aligns the flex item on the cross-axis of the flex container.

Values

Value Description
auto The item inherits the align-items value of its parent.
flex-start The item is aligned at the start of the cross-axis.
flex-end The item is aligned at the end of the cross-axis.
center The item is centered along the cross-axis.
baseline The item is aligned such that its baseline aligns with the baseline of the flex container.
stretch The item is stretched to fill the container (default value).

Example

.container {
    display: flex;
    height: 200px;
    align-items: stretch; /* Default value */
    box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1);
    padding: 20px;
    border-radius: 10px;
    background-color: #fff;
    width: 80%;
    margin: auto;
}

.item {
    width: 100px;
    height: 50px;
    background-color: #f3f3f3;
    display: flex;
    align-items: center;
    justify-content: center;
    border-radius: 5px;
    margin: 10px;
}

.item-auto { align-self: auto; /* auto */ }
.item-flex-start { align-self: flex-start; /* flex-start */ }
.item-flex-end { align-self: flex-end; /* flex-end */ }
.item-center { align-self: center; /* center */ }
.item-baseline { align-self: baseline; /* baseline */ }
.item-stretch { align-self: stretch; /* stretch */ }

Description

The align-self property aligns a flex item within the flex container along the cross-axis. This property overrides the align-items value for the specific item. It is useful for changing the alignment of a single item when the other items in the container are aligned differently.

All Most Similar Properties

Property Description Difference
align-items Aligns flex items along the cross-axis of the flex container. align-items applies to all flex items within the container, whereas align-self applies to individual flex items.
justify-content Aligns flex items along the main-axis of the flex container. justify-content distributes items within a single line along the main axis, whereas align-self aligns items along the cross-axis within a single line.
align-content Aligns a flex container's lines within the flex container when there is extra space on the cross-axis. align-content aligns the flex lines themselves within the container, whereas align-self aligns individual items within a single flex line.
