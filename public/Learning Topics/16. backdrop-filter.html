backdrop-filter

Definition

The backdrop-filter CSS property applies one or more filters to the backdrop of an element, allowing for effects like blurring or color shifting the area behind an element.

Values

Value Description
none No effect is applied.
blur(<length>) Applies a Gaussian blur to the backdrop.
brightness(<number> | <percentage>) Adjusts the brightness of the backdrop.
contrast(<number> | <percentage>) Adjusts the contrast of the backdrop.
grayscale(<number> | <percentage>) Converts the backdrop to grayscale.
hue-rotate(<angle>) Applies a hue rotation to the backdrop.
invert(<number> | <percentage>) Inverts the colors of the backdrop.
opacity(<number> | <percentage>) Adjusts the opacity of the backdrop.
saturate(<number> | <percentage>) Saturates the backdrop.
sepia(<number> | <percentage>) Applies a sepia filter to the backdrop.
url(<filter-function>) Applies an SVG filter from a URL.

Example

.example-none {
  backdrop-filter: none; /* No effect */
}

.example-blur {
  backdrop-filter: blur(5px); /* Applies a blur effect */
}

.example-brightness {
  backdrop-filter: brightness(150%); /* Increases brightness */
}

.example-contrast {
  backdrop-filter: contrast(200%); /* Increases contrast */
}

.example-grayscale {
  backdrop-filter: grayscale(100%); /* Converts to grayscale */
}

.example-hue-rotate {
  backdrop-filter: hue-rotate(90deg); /* Rotates hue by 90 degrees */
}

.example-invert {
  backdrop-filter: invert(100%); /* Inverts colors */
}

.example-opacity {
  backdrop-filter: opacity(50%); /* Reduces opacity */
}

.example-saturate {
  backdrop-filter: saturate(300%); /* Increases saturation */
}

.example-sepia {
  backdrop-filter: sepia(100%); /* Applies sepia filter */
}

Description

The backdrop-filter property is used to apply graphical effects such as blurring or color shifting to the area behind an element. This is particularly useful for creating frosted glass or other visually appealing effects in web design. The effects apply to everything behind the element, including text, images, and other elements.

All Most Similar Properties

Property Description Difference
filter Applies visual effects to the element itself filter applies to the element, while backdrop-filter applies to the backdrop of the element
opacity Adjusts the transparency of the element Only adjusts transparency, no other visual effects
background Sets the background of an element Changes the background without applying visual effects
box-shadow Adds shadow effects around an element Adds shadows but does not affect the backdrop
mix-blend-mode Specifies how an element’s content should blend with the background Blends the content with the background but does not apply filter effects
