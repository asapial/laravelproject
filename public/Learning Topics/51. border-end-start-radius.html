border-end-start-radius

Definition

The border-end-start-radius CSS property defines the radius of the border end-start corner of an element, considering the writing mode, directionality, and text orientation.

Values

Value Description
<length> A length value defining the radius (e.g., px, em).
<percentage> A percentage value defining the radius, relative to the corresponding dimension of the box.
initial Sets the property to its default value.
inherit Inherits the value from its parent element.
unset Resets the property to its natural value, either inherited or initial.

Example

/* Different border-end-start-radius values */
.example1 {
    border-end-start-radius: 10px; /* Length in px */
}

.example2 {
    border-end-start-radius: 50%; /* Percentage */
}

.example3 {
    border-end-start-radius: initial; /* Initial value */
}

.example4 {
    border-end-start-radius: inherit; /* Inherited value */
}

.example5 {
    border-end-start-radius: unset; /* Unset value */
}

Description

The border-end-start-radius property sets the rounding of the corner at the intersection of the inline-end and block-start borders of an element, considering the writing mode, directionality, and text orientation. This property helps in creating consistent and flexible layouts for internationalization by adapting to the writing mode and text direction.

All Most Similar Properties

Property Description Difference
border-radius Sets the radii for all four corners of an element. Affects all corners, not just the end-start corner.
border-start-start-radius Sets the radius for the start-start corner. Applies to the start-start corner, not the end-start corner.
border-start-end-radius Sets the radius for the start-end corner. Applies to the start-end corner, not the end-start corner.
border-end-end-radius Sets the radius for the end-end corner. Applies to the end-end corner, not the end-start corner.
