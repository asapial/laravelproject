background-clip

Definition

The background-clip CSS property specifies whether an element's background, either the color or image, extends underneath its border.

Values

Value Description
border-box The background extends to the outside edge of the border (default).
padding-box The background extends to the outside edge of the padding.
content-box The background extends only to the edge of the content box.
text The background is clipped to the foreground text.

Example

/* Example values for background-clip */
.element-border-box {
  background-clip: border-box; /* The background extends to the outside edge of the border */
}

.element-padding-box {
  background-clip: padding-box; /* The background extends to the outside edge of the padding */
}

.element-content-box {
  background-clip: content-box; /* The background extends only to the edge of the content box */
}

.element-text {
  background-clip: text; /* The background is clipped to the foreground text */
}

Description

The background-clip property controls the extent of the background area for an element. By default, the background extends underneath the element's border (border-box). This property allows the background to be clipped to the padding area (padding-box), the content area (content-box), or to the text area (text), creating various visual effects depending on the specified value.

All Most Similar Properties

Property Description Difference
border Sets the border of an element. The border property defines the border itself, whereas background-clip defines where the background extends.
background-origin Specifies the origin position of the background image. background-origin determines the starting point of the background image, while background-clip defines where it ends.
box-shadow Adds shadow effects around an element's frame. box-shadow adds a shadow effect but doesn't affect the clipping of the background.
