background-origin

Definition

The background-origin CSS property sets the background positioning area, determining the origin position from which the background image is placed.

Values

Value Description
border-box The background image starts from the border's edge.
padding-box The background image starts from the padding's edge.
content-box The background image starts from the content's edge.

Example

.element1 {
  background-origin: border-box; /* The background starts from the border's edge */
}

.element2 {
  background-origin: padding-box; /* The background starts from the padding's edge */
}

.element3 {
  background-origin: content-box; /* The background starts from the content's edge */
}

Description

The background-origin property specifies the area within the box model (border-box, padding-box, or content-box) that serves as the starting point for the background image. This affects the background's position, but not its size or repetition.

All Most Simillar Properties

Property Description Difference
background-clip Determines the painting area of the background. background-origin sets the positioning area, while background-clip sets the painting area.
background-position Sets the initial position of the background image. background-origin defines the origin of the position, while background-position sets the coordinates within that origin.
