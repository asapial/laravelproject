border-bottom-right-radius

Definition

The border-bottom-right-radius CSS property rounds the bottom-right corner of an element.

Values

Value Description
<length> A length value to define the radius, such as px, em, rem, etc.
<percentage> A percentage value to define the radius, relative to the element's dimensions.
inherit Inherits the radius value from its parent element.
initial Sets the property to its default value.
unset Resets the property to its natural value, which means it will act as inherit if it inherits, or initial if it does not.

Example

/* length values */
.example1 {
    border-bottom-right-radius: 10px; /* 10 pixels radius */
}

.example2 {
    border-bottom-right-radius: 1em; /* 1 em radius */
}

.example3 {
    border-bottom-right-radius: 5rem; /* 5 rem radius */
}

/* percentage value */
.example4 {
    border-bottom-right-radius: 50%; /* 50% radius */
}

/* inherit value */
.example5 {
    border-bottom-right-radius: inherit; /* inherits the radius from parent */
}

/* initial value */
.example6 {
    border-bottom-right-radius: initial; /* sets the property to its default value */
}

/* unset value */
.example7 {
    border-bottom-right-radius: unset; /* acts as inherit or initial based on context */
}

Description

The border-bottom-right-radius property is used to define the rounding of the bottom-right corner of an element's outer border edge. It can accept length values, percentage values, and some keyword values. When using length values, the radius is defined in absolute units like pixels, ems, or rems. When using percentage values, the radius is relative to the dimensions of the element.

All Most Similar Properties

Property Description Difference
border-top-left-radius Rounds the top-left corner of an element. Applies to the top-left corner instead of the bottom-right.
border-top-right-radius Rounds the top-right corner of an element. Applies to the top-right corner instead of the bottom-right.
border-bottom-left-radius Rounds the bottom-left corner of an element. Applies to the bottom-left corner instead of the bottom-right.
border-radius A shorthand property for setting all four border-*-radius properties. Applies to all corners instead of just the bottom-right.
border Sets all border properties in one declaration, including width, style, and color. Applies to all border aspects, not just the radius.
