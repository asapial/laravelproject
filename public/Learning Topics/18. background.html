background

Definition

The background CSS property is a shorthand for setting multiple background-related properties in one declaration. These can include background color, image, position, size, repeat, origin, clip, and attachment.

Values

Value Description
background-color Sets the background color of an element.
background-image Sets one or more background images for an element.
background-position Sets the initial position of a background image.
background-size Sets the size of the background image.
background-repeat Sets how background images are repeated.
background-origin Sets the background positioning area.
background-clip Sets the painting area of the background.
background-attachment Sets whether a background image is fixed or scrolls with the rest of the page.

Example

/* Example values for background */
.element {
  background: red; /* background-color */
  background: url('image.png'); /* background-image */
  background: url('image.png') no-repeat; /* background-image, background-repeat */
  background: url('image.png') center center; /* background-image, background-position */
  background: url('image.png') / cover; /* background-image, background-size */
  background: url('image.png') no-repeat center center / cover; /* background shorthand */
}

Description

The background shorthand property allows you to set multiple background properties in a single declaration. You can use it to define a background color, one or more background images, their positions, sizes, repetition behavior, and other related properties. This is particularly useful for compacting your CSS code and improving readability.

All Most Similar Properties

Property Description Difference
background-color Sets the background color of an element. Only sets the color, not images or other background-related properties.
background-image Sets one or more background images for an element. Only sets the image(s), not other background-related properties.
background-position Sets the initial position of a background image. Only sets the position, not other background-related properties.
background-size Sets the size of the background image. Only sets the size, not other background-related properties.
background-repeat Sets how background images are repeated. Only sets the repeat behavior, not other background-related properties.
background-origin Sets the background positioning area. Only sets the origin, not other background-related properties.
background-clip Sets the painting area of the background. Only sets the clipping area, not other background-related properties.
background-attachment Sets whether a background image is fixed or scrolls with the page. Only sets the attachment behavior, not other background-related properties.
