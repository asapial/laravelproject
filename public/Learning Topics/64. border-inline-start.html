border-inline-start

Definition

The border-inline-start CSS property is a shorthand property for setting the width, style, and color of the logical start border of an element in the inline direction, based on the element's writing mode, directionality, and text orientation.

Values

Value Description
<border-width> Specifies the width of the border (e.g., thin, medium, thick, or specific length)
<border-style> Specifies the style of the border (e.g., none, solid, dotted, dashed, etc.)
<border-color> Specifies the color of the border (e.g., color name, hex value, rgb, rgba, etc.)
initial Sets the property to its default value
inherit Inherits the property from its parent element
unset Resets the property to its natural value (either inherited or initial)

Example

/* width style color */
element1 {
  border-inline-start: 2px solid red; /* 2px wide solid red border */
}

/* width and style */
element2 {
  border-inline-start: thick dotted; /* Thick dotted border */
}

/* style and color */
element3 {
  border-inline-start: dashed blue; /* Dashed blue border */
}

/* only width */
element4 {
  border-inline-start: medium; /* Medium border */
}

/* only color */
element5 {
  border-inline-start: green; /* Green border */
}

/* initial */
element6 {
  border-inline-start: initial; /* Initial border */
}

/* inherit */
element7 {
  border-inline-start: inherit; /* Inherited border */
}

/* unset */
element8 {
  border-inline-start: unset; /* Unset border */
}

Description

The border-inline-start property is part of the CSS Logical Properties and Values specification, allowing authors to control the layout and style of their content in a writing-mode-independent way. This property sets the border on the start side of the inline dimension, considering the writing mode, directionality, and text orientation of the element. It is a shorthand for border-inline-start-width, border-inline-start-style, and border-inline-start-color.

All Most Similar Properties

Property Description Difference
border-left Sets the width, style, and color of the left border of an element Physical property, always refers to the left side
border-right Sets the width, style, and color of the right border of an element Physical property, always refers to the right side
border-inline-end Sets the width, style, and color of the end border in the inline direction Logical property, refers to the end of the inline direction
