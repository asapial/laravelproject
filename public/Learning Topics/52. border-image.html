border-image

Definition

The border-image CSS property allows you to use an image as the border around an element.

Values

Value Description
none No image border.
url() Specifies the path to the image to be used as the border.
number A number specifying how the image should be sliced.
percentage A percentage specifying how the image should be sliced.
auto Uses the intrinsic dimensions of the image for slicing.
fill Fills the center of the border image area with the image.
repeat Repeats the image to fill the border area.
stretch Stretches the image to fill the border area.
round Rounds the image to fill the border area.
space Adds space between the images to fill the border area.
initial Sets the property to its default value.
inherit Inherits the value from its parent element.
unset Resets the property to its natural value, either inherited or initial.

Example

/* Different border-image values */
.example1 {
    border-image: url(border.png) 30 30 round; /* Image with 30px slice and round repeat */
}

.example2 {
    border-image: url(border.png) 20% 20% stretch; /* Image with 20% slice and stretch repeat */
}

.example3 {
    border-image: none; /* No image border */
}

.example4 {
    border-image: initial; /* Initial value */
}

.example5 {
    border-image: inherit; /* Inherited value */
}

Description

The border-image property sets an image to be used as the border for an element. The image can be sliced into sections and can be manipulated in various ways (e.g., stretched, repeated, or rounded) to fit the border area. This property allows for greater flexibility and creativity in designing borders compared to traditional solid or dashed borders.

All Most Similar Properties

Property Description Difference
border Sets the style, width, and color of the border. Uses solid, dashed, or other predefined styles, not images.
border-image-source Sets the image to be used as the border. Only sets the image source, not the slicing or repetition.
border-image-slice Specifies how to slice the image. Only sets the slicing values, not the source or repetition.
border-image-width Sets the width of the border image. Only sets the width, not the source or slicing.
border-image-outset Sets the amount by which the border image is extended. Only sets the outset values, not the source or slicing.
border-image-repeat Specifies how the border image is repeated. Only sets the repeat behavior, not the source or slicing.
