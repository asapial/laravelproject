border-image-source

Definition

The border-image-source CSS property specifies the image to use as the border image of an element.

Values

Value Description
none No image is specified, thus no border image will be used.
<image> URL to the image file to be used as the border image.
inherit Inherits the value from its parent element.

Example

/* Example with none value */
.example-none {
  border-image-source: none; /* No image is used as the border */
}

/* Example with image URL value */
.example-image {
  border-image-source: url('border.png'); /* Uses the specified image as the border */
}

/* Example with inherit value */
.example-inherit {
  border-image-source: inherit; /* Inherits the value from the parent element */
}

Description

The border-image-source property defines the image to use as the border of an element. When specified, the image is used instead of the border-style properties. This property allows for custom border designs using images.

All Most Similar Properties

Property Description Difference
border-image-slice Specifies how to slice the border image. border-image-slice defines the inward slices of the border image; border-image-source defines the source image.
border-image-width Sets the width of the border image area. border-image-width controls the thickness of the border image; border-image-source defines the source image.
border-image-repeat Defines how the border image is repeated. border-image-repeat controls the tiling method of the border image; border-image-source defines the source image.
