border-block-start-style

Definition

The border-block-start-style CSS property sets the style of an element's logical block-start border, which corresponds to the top border in a horizontal writing mode.

Values

Value Description
none No border.
hidden Same as none, but used for border conflict resolution for table elements.
dotted A series of dots.
dashed A series of short line segments.
solid A single solid line.
double Two solid lines.
groove Looks as though it is carved into the page.
ridge Looks as though it is coming out of the page.
inset Looks as though it is embedded into the page.
outset Looks as though it is coming out of the page.
inherit Inherits the border style from its parent element.
initial Sets the border style to its default value.
unset Resets the border style to its inherited or initial value.

Example

/* <style> value */
.element1 {
  border-block-start-style: dotted; /* Sets the border style to dotted */
}

.element2 {
  border-block-start-style: dashed; /* Sets the border style to dashed */
}

.element3 {
  border-block-start-style: solid; /* Sets the border style to solid */
}

.element4 {
  border-block-start-style: double; /* Sets the border style to double */
}

.element5 {
  border-block-start-style: groove; /* Sets the border style to groove */
}

.element6 {
  border-block-start-style: ridge; /* Sets the border style to ridge */
}

.element7 {
  border-block-start-style: inset; /* Sets the border style to inset */
}

.element8 {
  border-block-start-style: outset; /* Sets the border style to outset */
}

.element9 {
  border-block-start-style: inherit; /* Inherits the border style from the parent */
}

.element10 {
  border-block-start-style: initial; /* Sets the border style to its default value */
}

.element11 {
  border-block-start-style: unset; /* Resets the border style to its inherited or default value */
}

Description

The border-block-start-style property is part of the CSS Logical Properties and Values module, which provides the ability to control layout through logical, rather than physical, direction and dimension mappings. This property allows for more flexible and intuitive design for different writing modes and orientations. It specifically sets the style of the block-start border of an element, which typically corresponds to the top border in left-to-right, top-to-bottom writing modes.

All Most Similar Properties

Property Description Difference
border-top-style Sets the style of the top border. Physical property, always targets the top border regardless of writing mode.
border-block-end-style Sets the style of the block-end border. Targets the block-end border, corresponds to the bottom border in a horizontal mode.
border-inline-start-style Sets the style of the inline-start border. Targets the inline-start border, which corresponds to the left border in LTR mode.
border-inline-end-style Sets the style of the inline-end border. Targets the inline-end border, which corresponds to the right border in LTR mode.
