background-blend-mode

Definition

The background-blend-mode CSS property sets how an element's background images should blend with each other and with the element's background color.

Values

Value Description
normal Default blending, no blending applied.
multiply Multiplies the background layers, resulting in a darker image.
screen Multiplies the inverse of the background layers, resulting in a lighter image.
overlay Multiplies or screens the background layers, depending on the background color.
darken Keeps the darkest pixels from the background layers.
lighten Keeps the lightest pixels from the background layers.
color-dodge Brightens the background layers to reflect the top layer.
color-burn Darkens the background layers to reflect the top layer.
hard-light Multiplies or screens the colors, depending on the top color value.
soft-light Darkens or lightens the colors, depending on the top color value.
difference Subtracts the darker colors from the lighter colors.
exclusion Similar to difference, but with lower contrast.
hue Applies the hue of the top layer while preserving the luminance and saturation of the bottom layers.
saturation Applies the saturation of the top layer while preserving the luminance and hue of the bottom layers.
color Applies the color of the top layer while preserving the luminance of the bottom layers.
luminosity Applies the luminosity of the top layer while preserving the hue and saturation of the bottom layers.

Example

/* Example values for background-blend-mode */
.element-normal {
  background-blend-mode: normal; /* Default blending */
}

.element-multiply {
  background-blend-mode: multiply; /* Darker blend */
}

.element-screen {
  background-blend-mode: screen; /* Lighter blend */
}

.element-overlay {
  background-blend-mode: overlay; /* Overlay blend */
}

Description

The background-blend-mode property allows you to blend the background layers of an element, including the background color and background images, using various blending modes. This can be useful for creating complex visual effects without the need for multiple layers of images in your HTML.

All Most Similar Properties

Property Description Difference
mix-blend-mode Specifies how an element's content should blend with the content behind it. Affects the blending of the entire element, not just the background layers.
opacity Sets the transparency level of an element. Affects the entire element's opacity, not just the blending of background layers.
filter Applies graphical effects like blur or color shift. Affects the visual appearance of the element, not specifically the blending of background layers.
