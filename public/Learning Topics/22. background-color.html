background-color

Definition

The background-color CSS property sets the background color of an element, either through a color name, HEX, RGB, RGBA, HSL, or HSLA value.

Values

Value Description
Color Names Predefined color names like red, blue, green, etc.
HEX Values Hexadecimal values like #ff0000, #00ff00, #0000ff, etc.
RGB Values RGB values like rgb(255, 0, 0), rgb(0, 255, 0), rgb(0, 0, 255), etc.
RGBA Values RGB values with alpha channel like rgba(255, 0, 0, 0.5), etc.
HSL Values HSL values like hsl(0, 100%, 50%), hsl(120, 100%, 50%), etc.
HSLA Values HSL values with alpha channel like hsla(0, 100%, 50%, 0.5), etc.
Transparent Sets the background color to be fully transparent.

Example

/* Example values for background-color */
.element-color-name {
  background-color: red; /* Using a color name */
}

.element-hex {
  background-color: #ff0000; /* Using a HEX value */
}

.element-rgb {
  background-color: rgb(255, 0, 0); /* Using an RGB value */
}

.element-rgba {
  background-color: rgba(255, 0, 0, 0.5); /* Using an RGBA value */
}

.element-hsl {
  background-color: hsl(0, 100%, 50%); /* Using an HSL value */
}

.element-hsla {
  background-color: hsla(0, 100%, 50%, 0.5); /* Using an HSLA value */
}

.element-transparent {
  background-color: transparent; /* Fully transparent background */
}

Description

The background-color property allows you to set the background color of an element. This can be specified using various color formats including color names, hexadecimal values, RGB/RGBA, and HSL/HSLA values. The transparency of the background color can be controlled using RGBA and HSLA values. This property is commonly used to enhance the visual appearance of elements and improve user experience.

All Most Similar Properties

Property Description Difference
background-image Sets the background image of an element. Specifies an image instead of a color for the background.
background-gradient Applies a gradient as the background image of an element. Creates a gradient background instead of a solid color.
color Sets the color of text within an element. Affects the text color, not the background color.
opacity Sets the transparency level of an element. Affects the entire element's opacity, not just the background color.
