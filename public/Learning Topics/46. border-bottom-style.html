border-bottom-style

Definition

The border-bottom-style CSS property sets the style of an element's bottom border.

Values

Value Description
none No border.
hidden Same as none, but used in table elements to avoid border conflict resolution.
dotted A series of dots.
dashed A series of short line segments.
solid A single solid line.
double Two parallel lines with space between them.
groove A 3D grooved border that looks as if it is carved into the page.
ridge A 3D ridged border that looks as if it is coming out of the page.
inset A 3D inset border that makes the element look embedded.
outset A 3D outset border that makes the element look like it is coming out.

Example

/* Different border-bottom styles */
.example1 {
    border-bottom-style: none; /* no bottom border */
}

.example2 {
    border-bottom-style: hidden; /* hidden bottom border */
}

.example3 {
    border-bottom-style: dotted; /* dotted bottom border */
}

.example4 {
    border-bottom-style: dashed; /* dashed bottom border */
}

.example5 {
    border-bottom-style: solid; /* solid bottom border */
}

.example6 {
    border-bottom-style: double; /* double bottom border */
}

.example7 {
    border-bottom-style: groove; /* grooved bottom border */
}

.example8 {
    border-bottom-style: ridge; /* ridged bottom border */
}

.example9 {
    border-bottom-style: inset; /* inset bottom border */
}

.example10 {
    border-bottom-style: outset; /* outset bottom border */
}

Description

The border-bottom-style property specifies the style of the bottom border of an element. It determines the visual appearance of the border, such as whether it is solid, dotted, dashed, or a 3D effect. This property does not affect the width or color of the border, which are set using the border-bottom-width and border-bottom-color properties, respectively.

All Most Similar Properties

Property Description Difference
border-top-style Sets the style of the top border of an element. Applies to the top border.
border-left-style Sets the style of the left border of an element. Applies to the left border.
border-right-style Sets the style of the right border of an element. Applies to the right border.
border-style A shorthand property for setting the style of all four borders. Applies to all borders simultaneously.
border A shorthand property for setting border width, style, and color. Applies to all aspects of the border.
