background-size

Definition

The background-size CSS property sets the size of the background image. It allows you to control whether the image is stretched, scaled, or constrained within the element.

Values

Value Description
auto Default value. The background image is displayed at its original size.
cover Scales the background image to be as large as possible so that the background area is completely covered by the image. The image is scaled to maintain its aspect ratio.
contain Scales the image as large as possible without cropping or stretching so that it fits within the element. The image is scaled to maintain its aspect ratio.
<length> Sets the width and height of the background image. Can use any CSS length unit (px, em, etc.).
<percentage> Sets the width and height of the background image in percentage of the containing element's size.

Example

/* Using keyword values */
background-size: auto; /* Original size */
background-size: cover; /* Cover entire area */
background-size: contain; /* Contain within area */

/* Using length values */
background-size: 100px 50px; /* 100px wide and 50px tall */
background-size: 50% 50%; /* 50% of the element's width and height */

/* Using a combination */
background-size: 100px auto; /* 100px wide, height auto */
background-size: auto 100px; /* Width auto, 100px tall */

Description

The background-size property in CSS defines the dimensions of the background image. It can be used to ensure the background image covers the entire element, fits within it without distortion, or is displayed at specific dimensions. The property can take keywords like cover and contain which automatically adjust the size while maintaining the aspect ratio of the image. Alternatively, specific lengths or percentages can be provided to define the exact size of the background image. When both width and height values are provided, the image is scaled to those dimensions. If only one value is provided, the other dimension is set to auto which preserves the aspect ratio.

All Most Similar Properties

Property Description Difference
background-image Specifies one or more background images for an element. Defines the image itself rather than its size.
background-position Sets the initial position of a background image. Controls position of background, not size.
background-repeat Sets how background images are repeated. Controls repetition of background, not its size.
background-attachment Sets whether a background image scrolls with the page or is fixed. Controls scroll behavior, not size.
background-origin Specifies the positioning area of the background images. Defines area of positioning, not the size of the background.
