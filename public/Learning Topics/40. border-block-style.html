border-block-style

Definition

The border-block-style CSS property sets the style of the logical block-level borders of an element, which correspond to the top and bottom borders in a horizontal writing mode.

Values

Value Description
none No border.
hidden Same as none, but when applied to a table cell, it prevents any border conflict.
dotted A series of dots.
dashed A series of short line segments.
solid A single solid line.
double Two solid lines.
groove Looks as if it were carved into the surface.
ridge Looks as if it were coming out of the surface.
inset Looks as if it were embedded in the surface.
outset Looks as if it were coming out of the surface.
inherit Inherits the border style from its parent element.
initial Sets the border style to its default value.
unset Resets the border style to its inherited or default value.

Example

/* <style> value */
.element1 {
  border-block-style: none; /* No border */
}

.element2 {
  border-block-style: dotted; /* Dotted border */
}

.element3 {
  border-block-style: dashed; /* Dashed border */
}

.element4 {
  border-block-style: solid; /* Solid border */
}

.element5 {
  border-block-style: double; /* Double border */
}

.element6 {
  border-block-style: groove; /* Groove border */
}

.element7 {
  border-block-style: ridge; /* Ridge border */
}

.element8 {
  border-block-style: inset; /* Inset border */
}

.element9 {
  border-block-style: outset; /* Outset border */
}

.element10 {
  border-block-style: inherit; /* Inherits the border style from the parent */
}

.element11 {
  border-block-style: initial; /* Sets the border style to its default value */
}

.element12 {
  border-block-style: unset; /* Resets the border style to its inherited or default value */
}

Description

The border-block-style property is part of the CSS Logical Properties and Values module, providing control over the styles of block-level borders in a writing mode-independent manner. This property allows for setting the styles of the block-start and block-end borders of an element, accommodating different writing modes and text orientations. It enhances flexibility in design by addressing the logical flow of content, rather than fixed physical directions.

All Most Similar Properties

Property Description Difference
border-top-style Sets the style of the top border. Physical property, always targets the top border regardless of writing mode.
border-bottom-style Sets the style of the bottom border. Physical property, always targets the bottom border regardless of writing mode.
border-block-start-style Sets the style of the block-start border. Targets the block-start border, corresponds to the top border in a horizontal mode.
border-block-end-style Sets the style of the block-end border. Targets the block-end border, corresponds to the bottom border in a horizontal mode.
