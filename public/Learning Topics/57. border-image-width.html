border-image-width

Definition

The border-image-width CSS property sets the width of the border image area.

Values

Value Description
<number> Specifies the width of the border image in pixels.
percentage Specifies the width of the border image as a percentage of the element's border width.
auto Automatically adjusts the width of the border image based on the image dimensions.
inherit Inherits the value from its parent element.

Example

/* Example with number value */
.example-number {
  border-image-width: 30px; /* Sets the border image width to 30 pixels */
}

/* Example with percentage value */
.example-percentage {
  border-image-width: 20%; /* Sets the border image width to 20% of the border width */
}

/* Example with auto value */
.example-auto {
  border-image-width: auto; /* Automatically adjusts the border image width */
}

/* Example with inherit value */
.example-inherit {
  border-image-width: inherit; /* Inherits the value from the parent element */
}

Description

The border-image-width property determines the width of the area where the border image is applied. This property can be set using various units, including pixels, percentages, and the auto keyword, allowing for flexible and responsive border designs.

All Most Similar Properties

Property Description Difference
border-width Sets the width of an element's border. border-width applies to solid borders; border-image-width applies to image borders.
border-image-slice Specifies how to slice the border image. border-image-slice controls the inward slicing; border-image-width controls the thickness.
border-image-outset Specifies the amount by which the border image area extends beyond the border box. border-image-outset extends the border image area; border-image-width defines the thickness.
