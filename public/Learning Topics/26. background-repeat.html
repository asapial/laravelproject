background-repeat

Definition

The background-repeat CSS property sets how background images are repeated. A background image can be repeated along the horizontal axis, the vertical axis, both axes, or not at all.

Values

Value Description
repeat The background image is repeated both vertically and horizontally.
repeat-x The background image is repeated only horizontally.
repeat-y The background image is repeated only vertically.
no-repeat The background image is not repeated.
space The background image is repeated as much as possible without clipping, and the remaining space is distributed around the images.
round The background image is repeated and resized to fill the container without clipping.
initial Sets this property to its default value.
inherit Inherits this property from its parent element.

Example

/* Using keyword values */
background-repeat: repeat; /* Repeats both horizontally and vertically */
background-repeat: repeat-x; /* Repeats only horizontally */
background-repeat: repeat-y; /* Repeats only vertically */
background-repeat: no-repeat; /* Does not repeat */

/* Using additional values */
background-repeat: space; /* Repeats without clipping, space around images */
background-repeat: round; /* Repeats and resizes to fill container */

Description

The background-repeat property in CSS controls the repetition of background images. By default, background images are repeated both vertically and horizontally, filling up the entire container. This property allows you to change that behavior. You can set the image to repeat in one direction only, not repeat at all, or use special values like space and round to control how the images are distributed within the container. This property is useful for creating patterns or ensuring that a background image covers the entire element without gaps.

All Most Similar Properties

Property Description Difference
background-image Specifies one or more background images for an element. Defines the image itself rather than its repetition.
background-position Sets the initial position of a background image. Controls position of background, not repetition.
background-size Specifies the size of the background images. Defines size of background, not its repetition.
background-attachment Sets whether a background image scrolls with the page or is fixed. Controls scroll behavior, not repetition.
background-origin Specifies the positioning area of the background images. Defines area of positioning, not repetition.
