border-image-outset

Definition

The border-image-outset CSS property specifies the amount by which the border image area extends beyond the border box.

Values

Value Description
<length> Specifies the width of the border image area in length units.
<number> A unitless number that is multiplied with the border-width to get the border image area.
inherit Inherits the value from its parent element.

Example

/* Example with length value */
.example-length {
  border-image-outset: 10px; /* 10 pixels of border image area beyond the border box */
}

/* Example with number value */
.example-number {
  border-image-outset: 2; /* Border image area is twice the border-width */
}

/* Example with inherit value */
.example-inherit {
  border-image-outset: inherit; /* Inherits the value from the parent element */
}

Description

The border-image-outset property determines the amount by which the border image extends outside the border box. It can accept length values (e.g., pixels, ems) or a number that multiplies the border-width. This allows for precise control over the placement of border images beyond the content box.

All Most Similar Properties

Property Description Difference
border-image-width Sets the width of the border image area. border-image-width controls the thickness of the border image; border-image-outset controls the extension beyond the border box.
border-image-slice Specifies how to slice the border image. border-image-slice defines the inward slices of the border image; border-image-outset defines the outward extension.
border-image-repeat Defines how the border image is repeated. border-image-repeat affects the tiling of the border image; border-image-outset affects the outer extent of the border image.
