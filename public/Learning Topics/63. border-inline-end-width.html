border-inline-end-width

Definition

The border-inline-end-width CSS property sets the width of the logical end border of an element in the inline direction, based on the element's writing mode, directionality, and text orientation.

Values

Value Description
<length> Specifies the width using a length value (e.g., px, em, rem, etc.)
thin A thin border width
medium A medium border width (default)
thick A thick border width

Example

/* length */
element1 {
  border-inline-end-width: 2px; /* 2 pixels wide */
}

element2 {
  border-inline-end-width: 0.5em; /* 0.5 em wide */
}

/* thin */
element3 {
  border-inline-end-width: thin; /* Thin border */
}

/* medium */
element4 {
  border-inline-end-width: medium; /* Medium border */
}

/* thick */
element5 {
  border-inline-end-width: thick; /* Thick border */

Description

The border-inline-end-width property is part of the CSS Logical Properties and Values specification, which allows authors to control the layout and style of their content in a writing-mode-independent way. This property sets the width of the border at the end of the inline dimension, taking into account the writing mode, directionality, and text orientation of the element. It is a logical counterpart to the border-right-width or border-left-width properties in physical terms.

All Most Similar Properties

Property Description Difference
border-right-width Sets the width of the right border of an element Physical property, always refers to the right side
border-left-width Sets the width of the left border of an element Physical property, always refers to the left side
border-inline-start-width Sets the width of the start border in the inline direction Logical property, refers to the start of the inline direction
