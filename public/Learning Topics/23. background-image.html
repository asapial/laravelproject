background-image

Definition

The background-image CSS property sets one or more background images for an element. These images are drawn on top of the background color, and each image can be positioned, sized, and repeated in various ways.

Values

Value Description
url() Specifies the location of the image.
linear-gradient Defines a linear gradient as the background image.
radial-gradient Defines a radial gradient as the background image.
repeating-linear-gradient Defines a repeating linear gradient as the background image.
repeating-radial-gradient Defines a repeating radial gradient as the background image.
none No background image will be displayed.
multiple images You can specify multiple background images by separating them with commas, each image will be applied in the order they are listed.

Example

/* Example values for background-image */
.element-url {
  background-image: url('https://via.placeholder.com/300'); /* Using a URL */
}

.element-linear-gradient {
  background-image: linear-gradient(to right, red, yellow); /* Using a linear gradient */
}

.element-radial-gradient {
  background-image: radial-gradient(circle, red, yellow, green); /* Using a radial gradient */
}

.element-repeating-linear-gradient {
  background-image: repeating-linear-gradient(45deg, red, yellow 10%, green 20%); /* Using a repeating linear gradient */
}

.element-repeating-radial-gradient {
  background-image: repeating-radial-gradient(circle, red, yellow 10%, green 20%); /* Using a repeating radial gradient */
}

.element-none {
  background-image: none; /* No background image */
}

.element-multiple {
  background-image: url('https://via.placeholder.com/300'), linear-gradient(to right, red, yellow); /* Multiple images */
}

Description

The background-image property allows you to set an image or a gradient as the background of an element. You can specify one or multiple images, which will be layered on top of each other. This property accepts different types of images, including URL-based images, linear gradients, radial gradients, and repeating gradients. Each type provides different visual effects, and they can be combined to create complex backgrounds.

All Most Similar Properties

Property Description Difference
background-color Sets the background color of an element. Specifies a solid color instead of an image or gradient.
background-repeat Specifies how background images are repeated. Controls the repetition of the background image, not the image itself.
background-position Sets the starting position of a background image. Controls the position of the background image, not the image itself.
background-size Specifies the size of the background images. Controls the size of the background image, not the image itself.
