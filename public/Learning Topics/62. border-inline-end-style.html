border-inline-end-style

Definition

The border-inline-end-style CSS property sets the style of the logical end border of an element in the inline direction, based on the element's writing mode, directionality, and text orientation.

Values

Value Description
none No border
hidden Same as none, but used when border conflict resolution is required
dotted A series of dots
dashed A series of short lines
solid A single solid line
double Two solid lines
groove Looks as though it were carved in the canvas
ridge Looks as though it were coming out of the canvas
inset Looks as though the content is embedded in the canvas
outset Looks as though the content is coming out of the canvas

Example

/* none */
element1 {
  border-inline-end-style: none; /* No border */
}

/* hidden */
element2 {
  border-inline-end-style: hidden; /* Hidden border */
}

/* dotted */
element3 {
  border-inline-end-style: dotted; /* Dotted border */
}

/* dashed */
element4 {
  border-inline-end-style: dashed; /* Dashed border */
}

/* solid */
element5 {
  border-inline-end-style: solid; /* Solid border */
}

/* double */
element6 {
  border-inline-end-style: double; /* Double border */
}

/* groove */
element7 {
  border-inline-end-style: groove; /* Grooved border */
}

/* ridge */
element8 {
  border-inline-end-style: ridge; /* Ridged border */
}

/* inset */
element9 {
  border-inline-end-style: inset; /* Inset border */
}

/* outset */
element10 {
  border-inline-end-style: outset; /* Outset border */

Description

The border-inline-end-style property is part of the CSS Logical Properties and Values specification, which allows authors to control the layout and style of their content in a writing-mode-independent way. This property sets the style of the border at the end of the inline dimension, taking into account the writing mode, directionality, and text orientation of the element. It is a logical counterpart to the border-right-style or border-left-style properties in physical terms.

All Most Similar Properties

Property Description Difference
border-right-style Sets the style of the right border of an element Physical property, always refers to the right side
border-left-style Sets the style of the left border of an element Physical property, always refers to the left side
border-inline-start-style Sets the style of the start border in the inline direction Logical property, refers to the start of the inline direction
