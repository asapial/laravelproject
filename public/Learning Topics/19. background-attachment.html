background-attachment

Definition

The background-attachment CSS property sets whether a background image scrolls with the rest of the page or is fixed.

Values

Value Description
scroll The background image scrolls with the rest of the content.
fixed The background image is fixed with regard to the viewport.
local The background image scrolls with the element's contents.

Example

/* Example values for background-attachment */
.element-scroll {
  background-attachment: scroll; /* The background image scrolls with the content */
}

.element-fixed {
  background-attachment: fixed; /* The background image is fixed in the viewport */
}

.element-local {
  background-attachment: local; /* The background image scrolls with the element's contents */
}

Description

The background-attachment property controls the scrolling behavior of the background image. When set to scroll, the background image will move as the user scrolls. When set to fixed, the background image will stay in place, creating a parallax effect. The local value makes the background image move with the content of the element to which it is applied.

All Most Similar Properties

Property Description Difference
background-position Sets the initial position of a background image. Controls position, not attachment behavior.
background-repeat Sets how background images are repeated. Controls repetition, not attachment behavior.
background-size Sets the size of the background image. Controls size, not attachment behavior.
background A shorthand property for setting all background properties at once. Can include background-attachment, but also sets other background properties.
