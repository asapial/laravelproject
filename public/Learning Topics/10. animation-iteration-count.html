animation-iteration-count

Definition

The animation-iteration-count CSS property specifies the number of times an animation cycle should be played.

Values

Value Description
<number> A positive number specifying how many times the animation should be played.
infinite The animation will repeat forever.

Example

/* 1 iteration */
.example-one {
  animation-iteration-count: 1; /* The animation will play once */
}

/* 3 iterations */
.example-three {
  animation-iteration-count: 3; /* The animation will play three times */
}

/* infinite iterations */
.example-infinite {
  animation-iteration-count: infinite; /* The animation will play indefinitely */
}

Description

The animation-iteration-count property allows you to define how many times an animation should repeat. It accepts positive numbers or the keyword infinite to make the animation loop indefinitely. This property is useful for creating animations that need to repeat a specific number of times or continue without stopping.

All Most Similar Properties

Property Description Difference
animation-duration Specifies the length of time an animation takes to complete one cycle. Controls the duration of each animation cycle, not the number of cycles.
animation-timing-function Specifies the speed curve of an animation. Controls the pace of the animation, not the number of iterations.
