border-block-width

Definition

The border-block-width CSS property sets the width of an element's logical block-level borders, which correspond to the top and bottom borders in a horizontal writing mode.

Values

Value Description
<length> Specifies the width of the border in length units (e.g., px, em, rem).
thin A thin border width.
medium A medium border width (default).
thick A thick border width.
inherit Inherits the border width from its parent element.
initial Sets the border width to its default value.
unset Resets the border width to its inherited or default value.

Example

/* <style> value */
.element1 {
  border-block-width: thin; /* Thin border width */
}

.element2 {
  border-block-width: medium; /* Medium border width */
}

.element3 {
  border-block-width: thick; /* Thick border width */
}

.element4 {
  border-block-width: 2px; /* 2px border width */
}

.element5 {
  border-block-width: 0.5em; /* 0.5em border width */
}

.element6 {
  border-block-width: inherit; /* Inherits the border width from the parent */
}

.element7 {
  border-block-width: initial; /* Sets the border width to its default value */
}

.element8 {
  border-block-width: unset; /* Resets the border width to its inherited or default value */
}

Description

The border-block-width property is part of the CSS Logical Properties and Values module. It allows you to set the widths of the block-level borders (top and bottom) of an element in a writing mode-independent manner. This property is useful for creating layouts that need to be adaptable to different writing modes and text directions. By using logical properties, you can ensure that your design is more flexible and responsive to changes in writing modes.

All Most Similar Properties

Property Description Difference
border-top-width Sets the width of the top border. Physical property, always targets the top border regardless of writing mode.
border-bottom-width Sets the width of the bottom border. Physical property, always targets the bottom border regardless of writing mode.
border-block-start-width Sets the width of the block-start border. Targets the block-start border, corresponds to the top border in a horizontal mode.
border-block-end-width Sets the width of the block-end border. Targets the block-end border, corresponds to the bottom border in a horizontal mode.
