border-image-slice

Definition

The border-image-slice CSS property divides the image specified by border-image-source into regions. These regions are then used to form the components of an element's border image.

Values

Value Description
<number> Defines the offsets that slice the image inward from the top, right, bottom, and left edges.
percentage A percentage value to slice the border image.
fill Indicates that the center part of the border image should be preserved.
inherit Inherits the value from its parent element.

Example

/* Example with number value */
.example-number {
  border-image-slice: 30; /* Slices the image 30 pixels inward from each edge */
}

/* Example with percentage value */
.example-percentage {
  border-image-slice: 30%; /* Slices the image 30% inward from each edge */
}

/* Example with fill value */
.example-fill {
  border-image-slice: 30 fill; /* Slices the image 30 pixels inward and preserves the center */
}

/* Example with inherit value */
.example-inherit {
  border-image-slice: inherit; /* Inherits the value from the parent element */
}

Description

The border-image-slice property determines how to divide the border image into regions. It specifies how far inward the image is sliced from its outer edges. The resulting pieces form the sides and corners of the border. This property allows for detailed control over how border images are applied, enabling complex and visually rich designs.

All Most Similar Properties

Property Description Difference
border-image-width Sets the width of the border image area. border-image-width controls the thickness of the border image; border-image-slice controls the inward slicing.
border-image-outset Specifies the amount by which the border image area extends beyond the border box. border-image-outset extends the border image area; border-image-slice defines the inward slicing.
border-image-repeat Defines how the border image is repeated. border-image-repeat affects the tiling of the border image; border-image-slice affects the slicing.
