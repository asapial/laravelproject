border-bottom

Definition

The border-bottom CSS property sets the width, style, and color of an element's bottom border in a single declaration.

Values

Value Description
<border-width> Specifies the width of the bottom border.
<border-style> Specifies the style of the bottom border (e.g., solid, dashed, dotted).
<color> Specifies the color of the bottom border.

Example

/* <style> value */
.element1 {
  border-bottom: 2px solid black; /* 2px width, solid style, black color */
}

.element2 {
  border-bottom: thick dotted red; /* Thick width, dotted style, red color */
}

.element3 {
  border-bottom: thin dashed blue; /* Thin width, dashed style, blue color */
}

.element4 {
  border-bottom: medium double green; /* Medium width, double style, green color */
}

.element5 {
  border-bottom: 5px groove gray; /* 5px width, groove style, gray color */
}

Description

The border-bottom property is a shorthand for setting the individual border properties for the bottom border of an element. This shorthand property allows you to set the border width, style, and color in a single declaration, making your CSS code more concise and easier to read. The border is drawn outside the padding and content of the element, but inside the margin.

All Most Similar Properties

Property Description Difference
border-top Sets the width, style, and color of the top border. Targets the top border instead of the bottom border.
border-left Sets the width, style, and color of the left border. Targets the left border instead of the bottom border.
border-right Sets the width, style, and color of the right border. Targets the right border instead of the bottom border.
border-bottom-width Sets the width of the bottom border. Only sets the width, not the style or color.
border-bottom-style Sets the style of the bottom border. Only sets the style, not the width or color.
border-bottom-color Sets the color of the bottom border. Only sets the color, not the width or style.
