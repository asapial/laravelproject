border-block-end

Definition

The border-block-end CSS property is a shorthand property for setting the individual logical block-end border properties in a single declaration, including border-block-end-width, border-block-end-style, and border-block-end-color.

Values

Value Description
<border-width> Specifies the width of the block-end border.
<border-style> Specifies the style of the block-end border (e.g., solid, dashed, dotted).
<border-color> Specifies the color of the block-end border.

Example

/* Using all values */
border-block-end: 3px solid red; /* 3px width, solid style, red color */

/* Using width and style only */
border-block-end: 5px dotted; /* 5px width, dotted style */

/* Using width only */
border-block-end: 2px; /* 2px width */

/* Using style and color */
border-block-end: dashed blue; /* dashed style, blue color */

Description

The border-block-end property is a shorthand for setting the width, style, and color of the block-end border of an element. The block-end side is determined by the element's writing mode, which means it will be the bottom border in a horizontal-tb writing mode and the left border in a vertical-rl writing mode. This property allows for more intuitive styling in different writing modes.

All Most Similar Properties

Property Description Difference
border Shorthand for all four borders. Sets all borders, not just the block-end border.
border-bottom Shorthand for the bottom border. Always affects the bottom border.
border-left Shorthand for the left border. Always affects the left border.
border-block-start Shorthand for the block-start border properties. Affects the block-start border.
border-block-end-width Sets the width of the block-end border. Only sets the width of the block-end border.
border-block-end-style Sets the style of the block-end border. Only sets the style of the block-end border.
border-block-end-color Sets the color of the block-end border. Only sets the color of the block-end border.
