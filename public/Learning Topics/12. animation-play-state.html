animation-play-state

Definition

The animation-play-state CSS property specifies whether an animation is running or paused.

Values

Value Description
running The animation is currently running.
paused The animation is currently paused.

Example

/* Running */
.example-running {
  animation-play-state: running; /* The animation is running */
}

/* Paused */
.example-paused {
  animation-play-state: paused; /* The animation is paused */
}

Description

The animation-play-state property allows you to control whether an animation is running or paused. This can be useful for scenarios where you want to start, stop, or control the timing of animations dynamically. By default, animations are running. When set to paused, the animation will stop at the current frame and resume when set back to running.

All Most Similar Properties

Property Description Difference
animation-duration Specifies the length of time an animation takes to complete one cycle. Controls the duration of the animation, not its play state.
animation-iteration-count Specifies the number of times an animation should play. Controls how many times the animation plays, not whether it is running or paused.

    }
