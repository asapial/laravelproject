animation-timing-function

Definition

The animation-timing-function CSS property specifies the speed curve of an animation.

Values

Value Description
ease Starts slow, speeds up, then slows down at the end.
linear Constant speed from start to end.
ease-in Starts slow and speeds up.
ease-out Starts fast and slows down.
ease-in-out Starts slow, speeds up in the middle, then slows down at the end.
step-start Jumps immediately to the end state and stays there until the end.
step-end Maintains the start state until the end, then jumps to the end state.
steps(n, start/end) Divides the animation into n equal steps, where start or end determines the change point.
cubic-bezier(x1, y1, x2, y2) Defines a custom timing function with cubic Bézier curve, using control points (x1, y1, x2, y2).

Example

/* ease */
.example-ease {
  animation-timing-function: ease; /* Starts slow, speeds up, then slows down at the end */
}

/* linear */
.example-linear {
  animation-timing-function: linear; /* Constant speed from start to end */
}

/* ease-in */
.example-ease-in {
  animation-timing-function: ease-in; /* Starts slow and speeds up */
}

/* ease-out */
.example-ease-out {
  animation-timing-function: ease-out; /* Starts fast and slows down */
}

/* ease-in-out */
.example-ease-in-out {
  animation-timing-function: ease-in-out; /* Starts slow, speeds up in the middle, then slows down at the end */
}

/* step-start */
.example-step-start {
  animation-timing-function: step-start; /* Jumps immediately to the end state and stays there until the end */
}

/* step-end */
.example-step-end {
  animation-timing-function: step-end; /* Maintains the start state until the end, then jumps to the end state */
}

/* steps(4, start) */
.example-steps-start {
  animation-timing-function: steps(4, start); /* Divides the animation into 4 equal steps, changing at the start of each step */
}

/* steps(4, end) */
.example-steps-end {
  animation-timing-function: steps(4, end); /* Divides the animation into 4 equal steps, changing at the end of each step */
}

/* cubic-bezier(0.42, 0, 0.58, 1) */
.example-cubic-bezier {
  animation-timing-function: cubic-bezier(0.42, 0, 0.58, 1); /* Custom timing function using a cubic Bézier curve */
}

Description

The animation-timing-function property controls how an animation progresses through time by defining the speed curve. This can make an animation feel more natural by varying the speed at which it moves from start to end. The property can use predefined functions such as ease, linear, and ease-in-out, or custom functions like steps() and cubic-bezier(). This flexibility allows for precise control over the pacing of animations.

All Most Similar Properties

Property Description Difference
transition-timing-function Specifies the speed curve of a transition. Similar in function but used for CSS transitions instead of animations.
animation-duration Specifies the length of time an animation takes to complete one cycle. Controls the duration of the animation, not the pacing.
